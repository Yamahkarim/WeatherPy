Analysis

Purpose

Does it REALLY get warmer by the equator?

Though seemingly obvious, WeatherPy Global seeks to confirm the phenomenon by employing various Data Science tools and methods. This website illustrates the findings.

Process

Using the Python’s Pandas library, a random set over of 1500 latitudes and longitudes were created to compile a list of random cities from across the globe.

The Open Weather API was employed to call upon weather data for these cities.

4 different weather factors were compared to respective latitude coordinates for analysis: Maximum Temperature, Cloudiness, Humidity, and Wind Speed.

This data was then compiled into a CSV file and can be viewed here

Finally, Python’s MatPlotLib library was used to visualize the results.

Results

As expected, the temperature in this sample increased with proximity to the equator.

Our analysis unveils that humidity around the globe peaks around the equator as well.

In regard to cloudiness versus latitude, an obvious correlation does not exist. It is interesting to note, however, that there is a strong concentration of cities sitting at 0, 80, and 100% cloudiness.
